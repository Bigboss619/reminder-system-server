Backend Installations
1. Express (API Server) = npm install express
2. Supabase Client (DB + Auth access) = npm install @supabase/supabase-js
3. Dotenv (Environment variables) = npm install dotenv
4. CORS (Allow frontend requests) = npm install cors
5. Nodemon (Dev auto restart) = npm install -D nodemon
6. XLSX (Excel batch uploads) = npm install xlsx
7. Multer (File uploads) = npm install multer
8. Zod (Validate API input) = npm install zod
9. UUID (Generate IDs) = npm install uuid
10. Winston (Production logs) = npm install winston
